## RAG pdf

Chatbot construit avec chainlit qui réponds à vos question sur les pdf que vous lui donnez.

### Technologies

- **RAG** :
	- ChromaDB 
	- ParentDocumentRetriever 
	- RecursiveCharacterTextSplitter

- **Modèles** :
	- llama3:instruct
	- hkunlp/instructor-large

- **Interface** :
	- chainlit

### Pour commencer
 1. `chainlit run main.py`
